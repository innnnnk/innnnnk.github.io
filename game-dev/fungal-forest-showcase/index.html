背景

Project_DND 目前卡在场景搭建阶段。Demo 需要一个能展示核心玩法（自由探索 + 回合制战斗 + 2.5D 美术风格）的展示场景。

场景主题：真菌森林（Fungal Forest）。

空间结构：珠串式布局

Demo 场景采用线性珠串结构——6 个精心编排的区域沿一条主路径串联，而非开放世界：

1
2

🌿入口 → 🍄走廊 → ⚔小战 → 🌑峡谷 → 👑BOSS → ✨结尾
 好奇     沉浸     紧张     压抑     高潮     释放

为什么不是开放区域？

  1. 引导可控 — 确保 Demo 玩家在 60 分钟内看到所有精彩内容
  2. 资产需求低 — 不需要填充 360° 视野，只需打磨可见路径
  3. 节奏可控 — “下一个转角有什么"比"随便逛逛"更有驱动力

D 深菇峡谷 战斗后喘息，氛围升级 ~30m 直廊 ~60s 压抑
E 巨菇王座 BOSS 战（真菌之主 + 2 小怪） 30×25m 圆形 ~180s 高潮

设计要点

空间节奏：同一条路，六种感受

核心技巧是改变宽度、曲率和围合感：

  • A 入口 — 开阔椭圆，给人安全感
  • B 走廊 — S 形弯道，激发好奇心（“拐角后面是什么？"）
  • C 空地 — 突然开阔 + 障碍物 → 警觉 → 战斗触发
  • D 峡谷 — 笔直收窄，两侧高墙，压抑→前方有光→必须前进
  • E 竞技场 — 圆形闭合空间，无处可逃
  • F 出口 — 重新开阔，暖光，胜利释放

光照梯度

从暖到冷再到暖的完整弧线：

  • A: 暖色夕阳 God Rays → B: 冷色荧光渐显 → C: 均匀中性战斗光
  • D: 最暗，纯荧光照明 → E: 戏剧化强对比 → F: 暖金色胜利光

Encounter 设计

小型遭遇（C 区）： 2 只远程真菌幼体 + 中央倒木掩体 → 教学性质的暖身战。同伴（矮人矿工）参与战斗，协助清小怪。

BOSS 战（E 区）： 真菌之主（近战 AOE + 召唤 + 回血）+ 第 2 回合孵化 2 只小怪 + 动态孢子云（提供可移动隐形掩体）。Boss 战为四阶段动态结构——详见下方「Boss 战四阶段设计」。

玩家角色：叙事滤镜方案

Demo 提供三种固定身份 + 一个自定义，共享同一套关卡流程（A→B→C→D→E→F），仅在文本/对话上产生差异。玩家通过入口处的简单选择确定身份。

身份 种族 一句话描述 与矮人同伴的互动差异
调查员 人类 佣兵公会派遣。任务：调查矿工失踪 同伴对他有戒备——“你是公会的人，你们只在乎矿石。“释放后态度软化：“但你没有丢下我。”
矿工 矮人 迟到的矿工——或幸存者。进入森林时他的队伍已经进去了 同伴认出他。劫后余生的亲切感——但也因此更容易触发反水（情感绑定越强→感染越容易深入）。释放后同伴说：“你来了。我们都在等你来。”
精灵学者 精灵 来自工业城市。来研究森林生态——不知道下面是矿场 同伴对这个"外人"不信任——“你在研究我们？像研究标本？“释放后态度转变：“你不是来研究。你是来……理解的。”
自定义 任意 三方雇佣兵。不受任何势力绑定 中性关系。同伴对雇佣兵保持职业距离，释放后简单点头——“欠你一条命。”

设计原则： 不是"独立故事线”（DOS2 式工作量 ×3），而是**“叙事滤镜”**（同一流程 + 文本条件分支 ≈ ×1.3 工作量）。Game.csv 中加 Role 条件列即可实现。

真菌之主的叙事身份

它是第一滴源脉之血凝固后、与地底真菌结合而成的原始意识。比螺旋文明更古老。在人类、精灵、矮人出现之前——它就在地底沉睡。没有语言、没有思想——只有一种缓慢的、跨越千年的感知。它记得巨兽的心跳。

矿工触发了什么： 矿工挖到了一片穿透真菌之主沉睡层的矿脉。在这个深度，矿石和菌丝互溶。触碰矿石 = 触碰它。矿工开始异化，不是被寄生——是被吸入它的记忆网络。

真菌之主 vs 矿工的关系： 真菌之主只有一个。矿工不会变成新的真菌之主——他们变成网络的节点。他们的记忆还在，他们通过真菌之主继续存在。BOSS 战不是与一只怪物的战斗——是与一个由矿工记忆编织成的存在战斗。

击败的意义： 击败不是消灭——是暂时切断物理锚点。它在更深处继续存在。晶核是它的一块记忆碎片——它在脉动，在被带出森林后继续"记得"你。

Demo 元素 与真菌之主的关系
矿工日记 矿工被网络"记住"时的第一手记录
菌丝宿主 异化中期——正在变成网络的节点
螺旋纹路 螺旋文明后来发现它，试图标记/控制

Boss 掉落的铭牌 被刮掉名字的身份牌。和矿工的铭牌同一个模具——但比这十二人更早

收集物：矿工铭牌

设计目的

玩家在 Demo 中一路捡到的矿工身份铭牌，在 F 区汇聚成一个核心矛盾：委托人派了十二个矿工，但铭牌数量是十三。铭牌不是"收集品”——是"证据链"的物证载体。

分布

位置 数量 说明
B 区营地残骸 3 块 散落在帐篷周围，部分被菌丝半掩
C 区（战后） 2 块 两个菌丝宿主被击败后，干壳旁各掉落一块
D 区峡谷 2 块 路边——像是匆忙中掉落的
E 区 Boss 场地 3 块 散布在场地上
E 区 Boss 掉落 1 块 真菌之主消散后，地上亮着一块——名字被刮掉了，铭牌微微发热
同伴脖子上 1 块 B 区营救时，玩家看到矮人脖子上挂着一块。不可拾取

合计：11（拾取）+ 1（Boss 掉落）+ 1（同伴脖子上看到）= 13

铭牌描述

  • 普通铭牌（统一描述，11 块全部相同）： 一块矿工身份牌。名字已模糊不清。
  • Boss 掉落的铭牌： 一块矿工身份牌。和其他十一块是同一个模具。不同的是——上面的名字被刮掉了。不是锈，是有人用东西刮的。触碰到时，微微发热。
  • 同伴脖子上的铭牌（B 区营救时的 UI 提示）： 他胸前挂着一块金属牌子，生了锈，上面隐约刻着一个名字。

设计要点

  1. 铭牌不区分名字——不需要给每块单独起名。名字模糊意味着"他们是谁不重要，重要的是数字对不上”
  2. C 区菌丝宿主掉落铭牌 → 玩家第一次将"捡铭牌"和"打菌丝宿主"联系起来：“铭牌 = 矿工 → 矿工 = 节点”
  3. Boss 掉落的铭牌是唯一特殊的——名字被刮掉 + 发热。暗示这块铭牌的主人比这十二人更早
  4. 同伴脖子上的铭牌不可拾取——只在 B 区营救时以 UI 提示形式出现一次。F 区同伴说出时，玩家会想起那一瞬间

菌丝网络的场景展现

矿工们没有消失——他们的身体被纳入了菌丝网络。以下是四个展现点，用菌丝、蘑菇和铭牌来暗示身体的存在和转化：

B 区：菌丝覆盖的身形

帐篷外，地面上有一团菌丝覆盖的隆起。菌丝的走势隐约勾出一个趴着的身形——肩膀、脊椎、向一侧弯折的膝盖。旁边的菌丝里半埋着一块铭牌。

C 区：菌丝宿主（已有）

战斗击杀 → 不倒地 → 孢子释放 → 躯体萎缩成干壳 → 2 秒后干壳上长出小蘑菇

D 区：岩壁菌丝脉络

峡谷岩壁上，菌丝沿着裂缝蔓延——不是附着，是嵌入。像石头自己长出了血管。几块铭牌被菌丝穿过——穿过那个刻了名字的位置。铭牌和菌丝已经长在一起了。

E 区：菌丝网络节点

场地中央的地面，所有菌丝脉络在这里收束。交汇处有几个浅浅的凹陷，大小和人形接近。菌丝在这些凹陷里长得最密——像是这里曾经有过什么，菌丝不舍得让它空着。

同伴：矮人矿工

设计动机

Boss 战需要"1 vs 真菌之主"之外的战术层次——一个同伴能在叙事和玩法上同时增加深度。

核心思路：同伴不是额外的战斗单位，而是 Demo 故事的一部分。 他在六区中拥有完整叙事弧线：被救→同战→被感染→反水→被释放→成为真相的指证者。

B 区 — 解救
  营地残骸旁。菌丝缠绕的矮人矿工奄奄一息。
  玩家靠近 → 交互解困。营救时相机拉近，看到矮人脖子上的铭牌。
  提示文本："他胸前挂着一块金属牌子，生了锈，上面隐约刻着一个名字。"
  同伴站起来，拍掉身上的孢子。
  → "还有……六个人在里面。"

C 区 — 并肩作战
  同伴参与小型遭遇战，协助清小怪。
  菌丝宿主被击败、孢子释放、躯体干枯→长蘑菇。干壳旁掉落两块铭牌。
  同伴沉默两秒："那是哈洛德。他上个月还在教我磨镐头。"

D 区 — 异常加深
  螺旋纹路附近，同伴的影子也短暂消失。
  → "我感觉……有人在摸我的后脑勺。"同伴回头看了一眼，说道："但没有人。"
  暗示：他比玩家更容易受影响——他是矿工，接触矿石更久。

E 区 — 感染→反水→释放（详见下方四阶段）
  真菌之主被击败后，地上掉落一块铭牌——名字被刮掉了。

F 区 — 指证
  委托人："你是第三个回来的。"
  同伴沉默。然后对委托人：
  → "当初不是只有十二个矿工进来吗？多出来的那个人是谁？"
  委托人站住了。转身离开。

阶段一：并肩（~60s）
  玩家 + 矮人 vs 真菌之主 + 2 小怪
  矮人负责清小怪、顶前排。一切正常。

阶段二：反水（~30s）
  触发条件：Boss 血量 < 60% 且 矮人在孢子云中停留超过 1 轮

  矮人头像框边缘出现菌丝纹路。
  他转身，动作帧从"攻击"切成"朝向玩家"。
  最后一句话：
  → "来吧，加入我/我们。"

  矮人暂时成为敌人单位，但比普通敌人弱：
  - HP 只有正常状态的 50%
  - AI 只用基础普攻，不用技能
  - 动作有延迟（菌丝操控 + 他在内心抵抗）

  玩家瞬间变成 1 vs 4（真菌之主 + 2 小怪 + 虚弱矮人）
  压力增加，但不是碾压——矮人故意被设计成虚弱状态

阶段三：释放（~20s）
  矮人的 HP 降到 0 → 不放"死亡红牌" → 触发特殊演出：

  菌丝出体 → 孢子散入空中 → 矮人倒地 → 起身，状态清除

  矮人跪在场地边缘，单手撑地。不再参战。
  他说：
  → "我看到了他们……哈洛德、老詹姆、卡里克……
      他们回不来了。"

  释放效果：
  → 真菌之主短暂失神 1 轮（失去了一个节点）
  → 玩家获得"觉醒"buff（攻击力 +30%，持续 2 轮）

阶段四：收尾（~70s）
  玩家 + buff vs 真菌之主 + 残余小怪
  矮人在边缘，活着，但不再参战。
  击败真菌之主 → 晶核掉落 + 一块铭牌（名字被刮掉，微微发热）

设计原理

为什么是"反水→释放→增益”？

  • 同伴反水时是虚弱的 → 数值上可控（HP 减半 + 无技能 + 动作延迟）
  • 击败他不是杀他 → 是"释放"他 → 和 C 区菌丝宿主的死亡规则一致
  • 释放效果帮助玩家 → 不是惩罚，而是奖励玩家"在乎同伴”
  • Buff 不是无中生有 → 来自真菌之主失去节点时的震荡

为什么不用能量条/大招系统？

  • Demo 开发时间紧，不做 5 分钟才会用一次的新系统
  • Boss 战才第一次用 → 玩家没学会就死了 → 挫败感

为什么不是纯剧情杀？

  • 玩家不能亲手打同伴的话，反水没有情感重量
  • 情感难度 > 数值难度——这是"你必须亲手打败你救的人”

A 入口 ── 委托人给任务。
          「十二个矿工，两周前进入的。找到他们——或者找到他们发生了什么。」
          三选一快速职业选择（战士 / 法师 / 游荡者）。
          玩家独自进入森林。

B 走廊 ── 营地残骸 + 矿工日记 + 菌丝覆盖的身形。
          「第三天。蘑菇在跟我说话……它似乎在……记住我。」（字迹中断）
          帐篷外：菌丝覆盖的隆起，隐约勾勒出趴着的身形。
          旁边菌丝里半埋着铭牌。搜索营地 → 拾取 3 块铭牌。
          营地残骸旁发现被困在菌丝中的矮人矿工——奄奄一息。
          交互解困 → 相机拉近 → 矮人脖子上的铭牌。
          提示文本："他胸前挂着一块金属牌子，生了锈，上面隐约刻着一个名字。"
          同伴起身，拍掉身上的孢子。
          →「还有……六个人在里面。」

C 空地 ── 两个菌丝宿主（被寄生的矿工）。
          同伴参与战斗，协助清小怪。
          战斗 —— AI 不躲避，朝玩家走来。
          击败后：不倒地，孢子从体内释放，躯体萎缩成干壳。
          干壳上长出小蘑菇。干壳旁各掉落一块铭牌。
          拾取 2 块铭牌。
          同伴沉默两秒："那是哈洛德。他上个月还在教我磨镐头。"

D 峡谷 ── 墙上螺旋纹路 + 岩壁菌丝脉络。
          菌丝嵌入岩壁裂缝，像石头长出了血管。
          几块铭牌被菌丝穿过——穿过那个刻了名字的位置。
          靠近纹路：影子消失 0.3 秒，嗡鸣频率改变。
          纹路每 6 秒微弱发光一次（脉搏感）。
          同伴的影子也短暂消失。
          →「我感觉……有人在摸我的后脑勺。」回头看了一眼,但没有人。
          拾取 2 块铭牌（路边）。

E BOSS ── 真菌之主。场地中央菌丝脉络收束为巨大节点，人形凹陷。
          Boss 战分为四个阶段：

          【阶段一：并肩（~60s）】
          玩家 + 矮人同伴 vs 真菌之主 + 2 小怪。
          矮人负责清小怪、顶前排。

          【阶段二：反水（~30s）】
          Boss 血量 < 60% 且同伴在孢子云中停留 > 1 轮时触发。
          同伴被感染 → 反水为敌人单位。
          HP 50%，只用普攻，动作有延迟（他在抵抗）。
          最后一句话："来吧，加入我。"

          【阶段三：释放（~20s）】
          同伴 HP → 0，不触发死亡红牌，改为"释放"演出：
          菌丝出体 → 孢子入空 → 倒地起身 → 状态清除。
          同伴退至场地边缘，不再参战。
          "我看到了他们……哈洛德、老詹姆、卡里克……
           他们回不来了。"
          释放效果：真菌之主失神 1 轮 + 玩家获得"觉醒"buff（攻击力 +30%，2 轮）。

          【阶段四：收尾（~70s）】
          玩家 + buff vs 真菌之主 + 残余小怪。
          击败后掉落「脉动的晶核」+ 一块铭牌（名字被刮掉，微微发热）。
          晶核描述：「此物在你手中脉动，仿佛记得什么。」
          铭牌描述：「一块矿工身份牌。和其他十一块是同一个模具。
          不同的是——上面的名字被刮掉了。不是锈，是有人用东西刮的。
          触碰到时，微微发热。」

F 出口 ── 委托人等在那里。
          「你是第三个回来的。前两个……记不太清了。」
          给报酬。袋子里有金币 + 一块微热的矿石。
          看晶核两秒：「留着吧。你应该留着。」转身离开。

          同伴（如果活着）：沉默。然后对委托人：
          「当初不是只有十二个矿工进来吗？多出来的那个人是谁？」
          委托人站住了。转身离开。

          身后蘑菇朝向微微转向玩家。
          嗡鸣消失。完全安静。然后风声。

全剧终。60 分钟。
