背景

Project_DND 目前卡在场景搭建阶段。Demo 需要一个能展示核心玩法（自由探索 + 回合制战斗 + 2.5D 美术风格）的展示场景。

场景主题：真菌森林（Fungal Forest）。

空间结构：珠串式布局

Demo 场景采用线性珠串结构——6 个精心编排的区域沿一条主路径串联，而非开放世界：

1
2

🌿入口 → 🍄走廊 → ⚔小战 → 🌑峡谷 → 👑BOSS → ✨结尾
 好奇     沉浸     紧张     压抑     高潮     释放

为什么不是开放区域？

  1. 引导可控 — 确保 Demo 玩家在 5-8 分钟内看到所有精彩内容
  2. 资产需求低 — 不需要填充 360° 视野，只需打磨可见路径
  3. 节奏可控 — “下一个转角有什么"比"随便逛逛"更有驱动力

六大区域

区域 代号 功能 空间尺寸 时长 情绪
A 森林入口 出生点，自由移动教学 15×15m 椭圆 ~60s 好奇
B 幽光走廊 氛围展示，S 形蜿蜒路径 ~50m 走廊 ~90s 沉浸
C 腐木空地 小型遭遇（2 只真菌幼体） 20×18m 椭圆 ~120s 紧张
D 深菇峡谷 战斗后喘息，氛围升级 ~30m 直廊 ~60s 压抑
E 巨菇王座 BOSS 战（真菌之主 + 2 小怪） 30×25m 圆形 ~180s 高潮
F 孢子之径 胜利感收束 10m 小径 ~30s 释放

设计要点

空间节奏：同一条路，六种感受

核心技巧是改变宽度、曲率和围合感：

  • A 入口 — 开阔椭圆，给人安全感
  • B 走廊 — S 形弯道，激发好奇心（“拐角后面是什么？"）
  • C 空地 — 突然开阔 + 障碍物 → 警觉 → 战斗触发
  • D 峡谷 — 笔直收窄，两侧高墙，压抑→前方有光→必须前进
  • E 竞技场 — 圆形闭合空间，无处可逃
  • F 出口 — 重新开阔，暖光，胜利释放

光照梯度

从暖到冷再到暖的完整弧线：

  • A: 暖色夕阳 God Rays → B: 冷色荧光渐显 → C: 均匀中性战斗光
  • D: 最暗，纯荧光照明 → E: 戏剧化强对比 → F: 暖金色胜利光

Encounter 设计

小型遭遇（C 区）： 2 只远程真菌幼体 + 中央倒木掩体 → 教学性质的暖身战

BOSS 战（E 区）： 真菌之主（近战 AOE + 召唤 + 回血）+ 第 2 回合孵化 2 只小怪 + 动态孢子云（提供可移动隐形掩体）
