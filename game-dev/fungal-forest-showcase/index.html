背景

Project_DND 目前卡在场景搭建阶段。Demo 需要一个能展示核心玩法（自由探索 + 回合制战斗 + 2.5D 美术风格）的展示场景。

场景主题：真菌森林（Fungal Forest）。

预计体验时长：30-35 分钟（关键路径叙事 ~730s + 两场战斗 + 探索走动）。

空间结构：珠串式布局

Demo 场景采用线性珠串结构——6 个精心编排的区域沿一条主路径串联，而非开放世界：

1
2

🌿入口 → 🍄走廊 → ⚔小战 → 🌑峡谷 → 👑BOSS → ✨结尾
 好奇     沉浸     紧张     压抑     高潮     释放

为什么不是开放区域？

  1. 引导可控 — 确保 Demo 玩家在 30-35 分钟内看到所有精彩内容
  2. 资产需求低 — 不需要填充 360° 视野，只需打磨可见路径
  3. 节奏可控 — “下一个转角有什么"比"随便逛逛"更有驱动力

设计要点

空间节奏：同一条路，六种感受

核心技巧是改变宽度、曲率和围合感：

  • A 入口 — 开阔椭圆，给人安全感。但出生点前方 8m 处有一棵倒下的蘑菇——菌盖朝向玩家来时的方向，不是朝上。这是玩家遇到的第一个"不对劲”。靠近按 E，角色底座阴影闪烁一次，根据身份说出不同台词。
  • B 走廊 — S 形弯道，激发好奇心（“拐角后面是什么？"）
  • C 空地 — 突然开阔 + 障碍物 → 警觉 → 战斗触发
  • D 峡谷 — 笔直收窄，两侧高墙，压抑→前方有光→必须前进。玩家静止时，底座阴影偶尔轻微抖动。
  • E 竞技场 — 圆形闭合空间，无处可逃
  • F 出口 — 重新开阔，暖光，但结尾不是"胜利释放”——是"你走出森林，数了数铭牌，十二块。你明明记得只捡了十一块。"

光照梯度

从暖到冷再到暖的完整弧线：

  • A: 暖色夕阳 God Rays → B: 冷色荧光渐显 → C: 均匀中性战斗光
  • D: 最暗，纯荧光照明 → E: 戏剧化强对比 → F: 暖金色胜利光

Encounter 设计

小型遭遇（C 区）： 2 只菌丝宿主 + 中央倒木掩体 → 教学性质的暖身战。同伴（矮人矿工）参与战斗，协助清小怪。

BOSS 战（E 区）： 真菌之主（近战 AOE + 召唤 + 回血）+ 第 2 回合孵化 2 只小怪 + 动态孢子云（提供可移动隐形掩体）。Boss 战为四阶段动态结构——详见下方「Boss 战四阶段设计」。

玩家角色：叙事滤镜方案

Demo 提供三种固定身份 + 一个自定义，共享同一套关卡流程（A→B→C→D→E→F），仅在文本/对话上产生差异。玩家通过入口处的简单选择确定身份。

身份 种族 一句话描述 与矮人同伴的互动差异
调查员 人类 佣兵公会派遣。任务：调查矿工失踪 同伴对他有戒备——“你是公会的人，你们只在乎矿石。“释放后态度软化：“但你没有丢下我。”
矿工 矮人 迟到的矿工——或幸存者。进入森林时他的队伍已经进去了 同伴认出他。劫后余生的亲切感——但也因此更容易触发反水（情感绑定越强→感染越容易深入）。释放后同伴说：“你来了。我们都在等你来。”
精灵学者 精灵 来自工业城市。来研究森林生态——不知道下面是矿场 同伴对这个"外人"不信任——“你在研究我们？像研究标本？“释放后态度转变：“你不是来研究。你是来……理解的。”
自定义 任意 三方雇佣兵。不受任何势力绑定 中性关系。同伴对雇佣兵保持职业距离，释放后简单点头——“欠你一条命。”

设计原则： 不是"独立故事线”（DOS2 式工作量 ×3），而是**“叙事滤镜”**（同一流程 + 文本条件分支 ≈ ×1.3 工作量）。Game.csv 中加 Role 条件列即可实现。

真菌之主的叙事身份

它是第一滴源脉之血凝固后、与地底真菌结合而成的原始意识。比螺旋文明更古老。在人类、精灵、矮人出现之前——它就在地底沉睡。没有语言、没有思想——只有一种缓慢的、跨越千年的感知。它记得巨兽的心跳。

矿工触发了什么： 矿工挖到了一片穿透真菌之主沉睡层的矿脉。在这个深度，矿石和菌丝互溶。触碰矿石 = 触碰它。矿工开始异化，不是被寄生——是被吸入它的记忆网络。

真菌之主 vs 矿工的关系： 真菌之主只有一个。矿工不会变成新的真菌之主——他们变成网络的节点。他们的记忆还在，他们通过真菌之主继续存在。BOSS 战不是与一只怪物的战斗——是与一个由矿工记忆编织成的存在战斗。

击败的意义： 击败不是消灭——是暂时切断物理锚点。它在更深处继续存在。晶核是它的一块记忆碎片——它在脉动，在被带出森林后继续"记得"你。

收集物：矿工铭牌

设计目的

玩家在 Demo 中一路捡到的矿工身份铭牌，在 F 区汇聚成一个核心矛盾：委托人派了十二个矿工，但铭牌数量是十三。铭牌不是"收集品”——是"证据链"的物证载体。

分布

位置 数量 说明
B 区营地残骸 3 块 散落在帐篷周围，部分被菌丝半掩
C 区（战后） 2 块 两个菌丝宿主被击败后，干壳旁各掉落一块
D 区峡谷 2 块 路边——像是匆忙中掉落的
E 区 Boss 场地 3 块 散布在场地上
E 区 Boss 掉落 1 块 真菌之主消散后，地上亮着一块——名字被刮掉了，铭牌微微发热
同伴脖子上 1 块 B 区营救时，玩家看到矮人脖子上挂着一块。不可拾取

合计：11（拾取）+ 1（Boss 掉落）+ 1（同伴脖子上看到）= 13

铭牌描述

  • 普通铭牌（统一描述，11 块全部相同）： 一块矿工身份牌。名字已模糊不清。
  • Boss 掉落的铭牌： 一块矿工身份牌。和其他十一块是同一个模具。不同的是——上面的名字被刮掉了。不是锈，是有人用东西刮的。触碰到时，微微发热。
  • 同伴脖子上的铭牌（B 区营救时的 UI 提示）： 他胸前挂着一块金属牌子，生了锈，上面隐约刻着一个名字。

设计要点

  1. 铭牌不区分名字——不需要给每块单独起名。名字模糊意味着"他们是谁不重要，重要的是数字对不上”
  2. C 区菌丝宿主掉落铭牌 → 玩家第一次将"捡铭牌"和"打菌丝宿主"联系起来：“铭牌 = 矿工 → 矿工 = 节点”
  3. Boss 掉落的铭牌是唯一特殊的——名字被刮掉 + 发热。暗示这块铭牌的主人比这十二人更早
  4. 同伴脖子上的铭牌不可拾取——只在 B 区营救时以 UI 提示形式出现一次。F 区同伴说出时，玩家会想起那一瞬间

菌丝网络的场景展现

矿工们没有消失——他们的身体被纳入了菌丝网络。以下是四个展现点，用菌丝、蘑菇和铭牌来暗示身体的存在和转化：

B 区：菌丝覆盖的身形

帐篷外，地面上有一团菌丝覆盖的隆起。菌丝的走势隐约勾出一个趴着的身形——肩膀、脊椎、向一侧弯折的膝盖。旁边的菌丝里半埋着一块铭牌。

C 区：菌丝宿主（已有）

战斗击杀 → 不倒地 → 孢子释放 → 躯体萎缩成干壳 → 2 秒后干壳上长出小蘑菇

D 区：岩壁菌丝脉络

峡谷岩壁上，菌丝沿着裂缝蔓延——不是附着，是嵌入。像石头自己长出了血管。几块铭牌被菌丝穿过——穿过那个刻了名字的位置。铭牌和菌丝已经长在一起了。

E 区：菌丝网络节点

场地中央的地面，所有菌丝脉络在这里收束。交汇处有几个浅浅的凹陷，大小和人形接近。菌丝在这些凹陷里长得最密——像是这里曾经有过什么，菌丝不舍得让它空着。

同伴：矮人矿工

设计动机

Boss 战需要"1 vs 真菌之主"之外的战术层次——一个同伴能在叙事和玩法上同时增加深度。

核心思路：同伴不是额外的战斗单位，而是 Demo 故事的一部分。 他在六区中拥有完整叙事弧线：被救→同战→被感染→反水→被释放→成为真相的指证者。

  螺旋纹路附近，同伴的影子也短暂消失。
  → "我感觉……有人在摸我的后脑勺。"同伴回头看了一眼，说道："但没有人。"
  暗示：他比玩家更容易受影响——他是矿工，接触矿石更久。

E 区 — 感染→反水→释放（详见下方四阶段）
  真菌之主被击败后，地上掉落一块铭牌——名字被刮掉了。

  委托人："你是第三个回来的。"
  同伴沉默。然后对委托人：
  → "当初不是只有十二个矿工进来吗？多出来的那个人是谁？"
  委托人站住了。转身离开。

  矮人暂时成为敌人单位，但比普通敌人弱：
  - HP 只有正常状态的 50%
  - AI 只用基础普攻，不用技能
  - 动作有延迟（菌丝操控 + 他在内心抵抗）

  玩家瞬间变成 1 vs 4（真菌之主 + 2 小怪 + 虚弱矮人）
  压力增加，但不是碾压——矮人故意被设计成虚弱状态

  矮人的 HP 降到 0 → 不放"死亡红牌" → 触发特殊演出：

  菌丝出体 → 孢子散入空中 → 矮人倒地 → 起身，状态清除

  矮人跪在场地边缘，单手撑地。不再参战。
  他说：
  → "我看到了他们……哈洛德、老詹姆、卡里克……
      他们回不来了。"

  释放效果：
  → 真菌之主短暂失神 1 轮（失去了一个节点）
  → 玩家获得"觉醒"buff（攻击力 +30%，持续 2 轮）

阶段四：收尾（~70s）
  玩家 + buff vs 真菌之主 + 残余小怪
  矮人在边缘，活着，但不再参战。
  击败真菌之主 → 晶核掉落 + 一块铭牌（名字被刮掉，微微发热）

设计原理

为什么是"反水→释放→增益”？

  • 同伴反水时是虚弱的 → 数值上可控（HP 减半 + 无技能 + 动作延迟）
  • 击败他不是杀他 → 是"释放"他 → 和 C 区菌丝宿主的死亡规则一致
  • 释放效果帮助玩家 → 不是惩罚，而是奖励玩家"在乎同伴"
  • Buff 不是无中生有 → 来自真菌之主失去节点时的震荡

为什么不用能量条/大招系统？

  • Demo 开发时间紧，不做 5 分钟才会用一次的新系统
  • Boss 战才第一次用 → 玩家没学会就死了 → 挫败感

为什么不是纯剧情杀？

  • 玩家不能亲手打同伴的话，反水没有情感重量
  • 情感难度 > 数值难度——这是"你必须亲手打败你救的人"

A 入口 ── 委托人给任务。
          「十二个矿工，两周前进入的。找到他们——或者找到他们发生了什么。」
          三选一快速职业选择（战士 / 游荡者。法师暂锁——法术系统 Demo 中不完整）。
          玩家独自进入森林。

          **出生点前方 8m，一棵倒下的蘑菇。**
          菌盖朝向玩家来时的方向——不是朝上，是朝路上。
          玩家靠近 → 按 E → 角色底座阴影闪烁一次（0.2 秒）。
          根据身份，角色说：
            - 调查员：「风是从那边吹来的。菌盖不应该朝这边。」
            - 矿工：「这不对。蘑菇知道有人要来。」
            - 精灵学者：「它在……转向？不，是生长方向。生长方向不会错。」
            - 自定义：（没有语音，只有底座阴影闪烁 + 轻微 UI 音效）
          
          机制影响：无。纯氛围。但从第 3 分钟开始，玩家知道"规则不对"。

B 走廊 ── 营地残骸 + 矿工日记 + 菌丝覆盖的身形。
          「第三天。蘑菇在跟我说话……它似乎在……记住我。」（字迹中断）
          帐篷外：菌丝覆盖的隆起，隐约勾勒出趴着的身形。
          旁边菌丝里半埋着铭牌。搜索营地 → 拾取 3 块铭牌。
          营地残骸旁发现被困在菌丝中的矮人矿工——奄奄一息。
          交互解困 → 相机拉近 → 矮人脖子上的铭牌。
          提示文本："他胸前挂着一块金属牌子，生了锈，上面隐约刻着一个名字。"
          同伴起身，拍掉身上的孢子。
          →「还有……六个人在里面。」

C 空地 ── 两个菌丝宿主（被寄生的矿工）。
          同伴参与战斗，协助清小怪。
          战斗 —— AI 不躲避，朝玩家走来。

          **同伴攻击菌丝宿主时，插入 0.3 秒"识别"动画。**
          同伴身体前倾（复用 Hit 帧），像在辨认敌人的脸。
          攻击音效延迟 0.1 秒——他在犹豫。
          如果击杀的是菌丝宿主，同伴不做欢呼动画，站在干壳前 Idle 2 秒。

          击败后：不倒地，孢子从体内释放，躯体萎缩成干壳。
          干壳上长出小蘑菇。干壳旁各掉落一块铭牌。
          拾取 2 块铭牌。

          **同伴台词（条件分支）：**
          - 默认：「那是哈洛德。他上个月还在教我磨镐头。」
          - 如果玩家已捡到 B 区哈洛德的铭牌：「哈洛德的牌子你捡到了……对吧？他……他以前总把牌子挂在脖子上，说怕丢了。」

D 峡谷 ── 墙上螺旋纹路 + 岩壁菌丝脉络。
          菌丝嵌入岩壁裂缝，像石头长出了血管。
          几块铭牌被菌丝穿过——穿过那个刻了名字的位置。
          靠近纹路：影子消失 0.3 秒，嗡鸣频率改变。
          纹路每 6 秒微弱发光一次（脉搏感）。
          同伴的影子也短暂消失。
          →「我感觉……有人在摸我的后脑勺。」回头看了一眼,但没有人。
          拾取 2 块铭牌（路边）。

E BOSS ── 真菌之主。场地中央菌丝脉络收束为巨大节点，人形凹陷。
          Boss 战分为四个阶段：

          【阶段一：并肩（~60s）】
          玩家 + 矮人同伴 vs 真菌之主 + 2 小怪。
          矮人负责清小怪、顶前排。

          【阶段二：反水（~30s）】
          Boss 血量 < 60% 且同伴在孢子云中停留 > 1 轮时触发。
          同伴被感染 → 反水为敌人单位。
          HP 50%，只用普攻，动作有延迟（他在抵抗）。
          最后一句话："来吧，加入我。"

          【阶段三：释放（~20s）】
          同伴 HP → 0，不触发死亡红牌，改为"释放"演出：
          菌丝出体 → 孢子入空 → 倒地起身 → 状态清除。
          同伴退至场地边缘，不再参战。
          "我看到了他们……哈洛德、老詹姆、卡里克……
           他们回不来了。"
          释放效果：真菌之主失神 1 轮 + 玩家获得"觉醒"buff（攻击力 +30%，2 轮）。

          【阶段四：收尾（~70s）】
          玩家 + buff vs 真菌之主 + 残余小怪。

          **Boss 战斗日志语言（不可理解化）：**
          - 孢子爆发：「真菌之主想起了疼痛。你感受到了 15 点。」
          - 护甲削弱：「真菌之主忘记了防御。它的外壳变得稀薄。」
          - 复制技能：「真菌之主正在学习你的攻击。它越来越像你了。」
          - 召唤小怪：「真菌之主想起了同伴。它们从地下爬了上来。」
          - 回血：「森林正在把它拉回去。」
          
          **孢子云"记忆"文本（淡金色，屏幕底部）：**
          - 第一次进入：「你闻到了……铁锈？不，是矿场。你从没来过这个矿场。」
          - 第二次进入：「这里有人喊过你的名字。但不是你的同伴。」
          - 第三次进入：「孢子云记得你。它比你先到这里。」
          不影响机制。纯氛围。

          **击败演出重构：**
          真菌之主血量归零 → **不播放死亡动画** → 静止不动。
          3 秒后，它缓慢沉入地下。
          没有"胜利"文本。同伴跪在场地边缘：「它走了。不是死了……是走了。」
          晶核不是在尸体上拾取的——玩家低头时，发现自己**脚边有一颗石头在脉动**。

          击败后掉落「脉动的晶核」+ 一块铭牌（名字被刮掉，微微发热）。
          晶核描述：「此物在你手中脉动，仿佛记得什么。」
          铭牌描述：「一块矿工身份牌。和其他十一块是同一个模具。
          不同的是——上面的名字被刮掉了。不是锈，是有人用东西刮的。
          触碰到时，微微发热。」

F 出口 ── 委托人等在那里。
          「你是第三个回来的。前两个……记不太清了。」
          给报酬。袋子里有金币 + 一块微热的矿石。

          同伴（如果活着）：不质问委托人。
          看着玩家，说：「晶核你留着吧。它……它在等你。」

          玩家走出森林，屏幕淡出。

          **黑屏 3 秒。**
          然后出现手写体文本（便签材质）：
          「你走出森林。包裹里的晶石还是热的。
          你数了数铭牌。十二块。
          你明明记得只捡了十一块。」

          然后才是 Steam 愿望单画面。

          身后蘑菇朝向微微转向玩家。
          嗡鸣消失。完全安静。然后风声。

全剧终。约 30-35 分钟。

边缘情况与体验补丁

以下是设计审查中发现的需要补充的边界情况。属于表现层面问题，可在开发中途逐步补齐，不阻塞白盒搭建。

补丁一：同伴死亡分支

当前叙事流程假设同伴全程存活。需设计一条"同伴在 C 区战死"的降级路径。

触发条件

C 区小型遭遇战中，同伴 HP 降至 0 → 不触发"释放"演出（那是 E 区专用）→ 同伴倒地，死亡红牌拍下。玩家可以继续战斗，战斗结算不变。

叙事降级

区域 同伴存活 同伴死亡
D 区 同伴：「我感觉……有人在摸我的后脑勺。」 跳过同伴台词。玩家独自走峡谷。菌丝纹路的光脉节奏比正常快半拍——像少了什么在吸收注意力。
E 区 四阶段完整（并肩→反水→释放→收尾） 跳过反水和释放阶段。直接 1 vs 真菌之主 + 2 小怪。Boss 血量下调 20%（少了同伴输出 + 少了释放 buff）。Boss 击败后晶核不变，但在 Boss 沉入地下后，玩家脚边除了晶核还多了一块铭牌——和 Boss 掉落那块位置不同，更靠近场地边缘，像被丢下的。
F 区 同伴指证委托人。 跳过同伴指证。委托人：「你是第二个回来的。第一个……记不太清了。」数量从"第三个"降为"第二个”——暗示同伴算第一个回来的人，但委托人忘了。微热的矿石仍然给玩家。

设计原则

  • 同伴死亡不惩罚玩家。Boss 血量下调 + 直接跳过反水阶段，整体难度持平甚至略低
  • 叙事不降级为"无趣"——同伴不在场时，D/E/F 区用环境细节填补缺失的台词
  • E 区多出的那块铭牌不计数（铭牌总数仍为 11+1+1=13）。它只是静静地在那里

补丁二：战斗教学触发器

A 区只教了自由探索移动（WASD）。C 区是第一场真正的遭遇战，中间缺少回合制战斗的教学桥。

方案：A 区出口前的"不会反击的敌人"

在 A 区即将进入 B 区的通道处（A 区边缘，玩家必经之路上），放置一只 孢子虫：

属性 内容
名称 孢子虫（Spore Mite）
外观 复用小型蘑菇模型，缩小至 0.3 倍，加一个缓慢上下浮动的动画。不需要新 Sprite
AI 无。不移动，不攻击。固定位置
HP 1
触发方式 玩家靠近到 3m 时弹出半透明 UI 提示

教学文本（分两段弹出）

第一段（靠近时）：

前方有一只孢子虫。按 Tab 进入战斗模式。

第二段（进入战斗后，玩家完成第一次移动）：

选择一个格子移动。靠近目标后，点击它进行攻击。

击败后：

  • 孢子虫萎缩（复用菌丝宿主死亡演出的简化版——只萎缩不长蘑菇）
  • 掉落一块铭牌（让玩家建立"战斗掉落 → 铭牌"的认知）
  • 铭牌描述和 B 区普通铭牌一致：一块矿工身份牌。名字已模糊不清。

设计要点

  • 不打断沉浸感。孢子虫是"世界里的生物"，不是"教学 NPC"
  • 不掉落不消失——如果玩家忽略它直接走进 B 区，它只是静静地浮动在那里
  • 铭牌总数从 11 拾取变为 12 拾取。但 F 区结尾文本不改——“你数了数铭牌。十二块。“教学战掉落的那块就是第十二块
  • 10 秒完成，不拖节奏

目前文档只有概念描述（“环境低频嗡鸣”、“靠近异常物频率改变”）。以下是落地所需的最少音频文件清单。

# 音频 用途 时长 优先级
1 森林环境底噪 A-F 全区循环。微风 + 偶尔的孢子爆裂声。温暖但带一丝不自然的空旷 60s 循环 P0
2 低频嗡鸣层 D/E 区叠加。40-60Hz，像地底深处传来的振动 30s 循环 P1

补丁五：玩家返回已通过区域的行为

线性珠串结构不锁死已通过区域的通道。玩家可以在任意时刻回头走。需要定义回退行为，避免出 bug。

规则

行为 规则
已拾取的铭牌 不重新生成。拾取位置保留空地面 Decal
已击败的菌丝宿主 干壳保留在原地（不消失）。蘑菇继续长在干壳上
已解救的同伴 跟随玩家。如果玩家从 C 区走回 B 区，同伴跟着走，不在营地处重新出现
已触发的叙事事件 日记不再弹出（已读过）。异常蘑菇不再触发底座阴影闪烁（已触发过）
已进入的 Boss 场地 Boss 战触发后 E 区入口不锁。但如果玩家在 Boss 战胜利后走回 E 区，只看到空的菌丝节点场地——菌丝不再发光，孢子云消散。地上的干壳还在
委托人 A 区和 F 区是同一个 NPC。如果玩家从 B/C/D 回到 A 区，委托人站在原地，只有一句通用台词：「调查进展如何？」（无剧情推进）

不需要的内容

  • 不需要回头路对话。不需要"你确定要往回走吗"提示。不需要隐藏区域
  • 目标是：回头走不出 bug，但不鼓励回头走

补丁六：结束画面的技术实现

当前描述：“黑屏 3 秒 → 手写体文本 → Steam 愿望单画面”。以下是技术落地细节。

实现方案

黑屏： Widget 全屏黑色背景，使用 UWidgetAnimation 做淡入（0→100% opacity，1.5s），而不使用关卡加载/卸载。保留当前关卡在背后渲染（性能无影响——F 区是 10m 小径，场景极简）。

手写体文本： 便签材质（复用现有 M_UI_StickyNote 材质实例），分三行显示，逐行淡入：

1
2
3
4

你走出森林。包裹里的晶石还是热的。

你数了数铭牌。十二块。
你明明记得只捡了十一块。
  • 第一行延迟 0s 淡入
  • 第二行延迟 3s 淡入
  • 第三行延迟 6s 淡入（和第二行一起显示——不覆盖，累积）
  • 全部文本显示完毕后停留 4 秒

愿望单画面： 文本淡出（1.5s）后，显示一张预渲染静态图：游戏标题 Logo + Steam 愿望单链接的 URL 文本（不是可点击的超链接——Demo 阶段通过 Steam overlay 的默认浏览器处理，或者直接显示文字"在 Steam 搜索 Project_DND 加入愿望单”）。不需要 Steam SDK 集成。

音效配合（补丁三 #8）： 黑屏时触发"嗡鸣消失 + 风起"音频。嗡鸣在第 2 秒淡出，风在第 2.5 秒进入。第一行文本出现时（延迟 0s），玩家同时听到风声——“不再有嗡鸣了”。

备选方案（更低成本）

如果 UI 动画开发时间不够：直接切成一张全屏静态图片（PNG），包含手写体文本 + 暗色渐变背景。玩家按任意键后显示愿望单页面。牺牲"逐行淡入"的效果，但不牺牲文本本身。
