设计原则

60 分钟版本不是"把 30 分钟版本拉长"，而是在完整主干上嫁接两条可选支线，让玩家有机会在主线之外"多待一会儿"——而这个"多待"的代价，是对世界的理解更深，但身体更不安。

“认知的代价"在 60 分钟版本中不只是主题，而是结构：你越想知道真相，你在森林里停留越久，你看到的"不对"就越多。

背景

Project_DND 目前卡在场景搭建阶段。Demo 需要一个能展示核心玩法（自由探索 + 回合制战斗 + 2.5D 美术风格）的展示场景。

场景主题：真菌森林（Fungal Forest）。

预计体验时长：30-60 分钟（主线 ~30min，全探索 ~60min）（关键路径叙事 ~730s + 两场战斗 + 探索走动）。

空间结构：珠串式布局

Demo 场景采用线性珠串结构——6 个精心编排的区域沿一条主路径串联，而非开放世界：

为什么不是开放区域？

  1. 引导可控 — 确保 Demo 玩家在 30-60 分钟（主线 ~30min，全探索 ~60min）内看到所有精彩内容
  2. 资产需求低 — 不需要填充 360° 视野，只需打磨可见路径
  3. 节奏可控 — “下一个转角有什么"比"随便逛逛"更有驱动力

设计要点

空间节奏：同一条路，六种感受

核心技巧是改变宽度、曲率和围合感：

  • A 入口 — 开阔椭圆，给人安全感。但出生点前方 8m 处有一棵倒下的蘑菇——菌盖朝向玩家来时的方向，不是朝上。这是玩家遇到的第一个"不对劲”。靠近按 E，角色底座阴影闪烁一次，根据身份说出不同台词。
  • B 走廊 — S 形弯道，激发好奇心（“拐角后面是什么？"）
  • C 空地 — 突然开阔 + 障碍物 → 警觉 → 战斗触发
  • D 峡谷 — 笔直收窄，两侧高墙，压抑→前方有光→必须前进。玩家静止时，底座阴影偶尔轻微抖动。
  • E 竞技场 — 圆形闭合空间，无处可逃
  • F 出口 — 重新开阔，暖光，但结尾不是"胜利释放”——是"你走出森林，数了数铭牌，十二块。你明明记得只捡了十一块。”

光照梯度

从暖到冷再到暖的完整弧线：

  • A: 暖色夕阳 God Rays → B: 冷色荧光渐显 → C: 均匀中性战斗光
  • D: 最暗，纯荧光照明 → E: 戏剧化强对比 → F: 暖金色胜利光

Encounter 设计

小型遭遇（C 区）： 2 只菌丝宿主 + 中央倒木掩体 → 教学性质的暖身战。同伴（矮人矿工）参与战斗，协助清小怪。

BOSS 战（E 区）： 真菌之主（近战 AOE + 召唤 + 回血）+ 第 2 回合孵化 2 只小怪 + 动态孢子云（提供可移动隐形掩体）。Boss 战为四阶段动态结构——详见下方「Boss 战四阶段设计」。

支线1：回声裂隙（Echo Cleft）

位置与入口

A 区异常蘑菇旁，一个被菌丝半掩的岩石裂隙。裂隙入口只有 1.5m 高，玩家需要弯腰进入——相机自动降低 0.5m，暗示"你在进入另一个空间"。

入口提示： A 区异常蘑菇交互后，如果玩家在周围逗留超过 5 秒，底座阴影会朝裂隙方向微微偏移一次，随后恢复正常。玩家可能注意到，也可能错过。

设计说明： 裂隙入口始终存在——不依赖异常蘑菇交互解锁。异常蘑菇的阴影偏移只是"提示"而非"钥匙"。玩家可以直接走进裂隙，错过蘑菇只意味着错过一个氛围细节，而非内容阻断。

空间

15×10m 的小型地下空间，天花板 2m，压抑。墙壁上有发光的菌丝脉络（复用 D 区岩壁材质）。中央有一个浅坑，坑底有积水。

敌人

敌人 数量 特性
菌丝幼体 3 只 比 C 区菌丝宿主弱（HP -30%），但会群体行动——一只发现玩家后，另外两只从侧面包抄
回声宿主 1 只 精英菌丝宿主。外观与普通宿主相同，但胸口嵌着一块发光的铭牌。额外技能：每 3 回合释放一次孢子云（范围 3×3 格，持续 2 回合）

可交互物件

1. 矿工画板（墙壁旁）

一块平整的岩壁，上面用菌丝画满了螺旋纹路。不是涂鸦——是重复的几何图案，像某种记录。

交互文本：「有人在这里画了很久。菌丝的走势精确得像测量过。」

叙事含义：这不是螺旋文明留下的纹路——是被异化的矿工自己画的。他们在网络里看到了某种图案，用菌丝把它描在了墙上。

2. 破损背包（浅坑边缘）

一个被菌丝半埋的矿工背包。

交互文本（第二篇矿工日记）：「第七天。我开始能听懂它们了。不是听懂话——是听懂意图。它们在……等待什么。或者说，等待谁。」

3. 回声宿主的铭牌（击败后掉落）

  • 拾取时的描述：「一块矿工身份牌。名字已模糊不清。」
  • 但当你把它放进背包时，铭牌计数器闪烁了一下——从 1 变成了 2，又变回 1。

这是视觉污染的第 0 次触发，比主线铭牌计数器（第 5 块时才抖动）更早。如果你先进入了回声裂隙，计数器在捡到第一块铭牌时就开始说谎。

视觉差异： 回声裂隙的闪烁是"快速一闪"（0.15 秒，数字瞬间跳变后恢复）。主线的第 11 块闪烁是"缓慢呼吸式"——数字从 11 呼吸到 12 再回 11，共三个呼吸周期（每个周期约 1.5 秒），颜色偏向淡紫。两次闪烁的视觉差异让玩家意识到"这次不一样"。

呼应"13 块铭牌"： 这块铭牌是"多出来的"那块——它和 Boss 掉落的那块被刮掉名字的铭牌是同一对，一个是"多出来的人"，另一个是"被遗忘的人"。

二周目种子： 回声宿主胸口的铭牌，在二周目（NG+）时名字不再模糊——显示的是玩家的名字。

击败回声宿主的演出

  • 不倒地。孢子从胸口（铭牌位置）涌出，躯体像"漏气的皮囊"一样萎缩。
  • 萎缩后，坑底的积水反射出微弱的光——光里有一个人形的倒影，但不是回声宿主的形状。倒影持续了 2 秒，然后消失。
  • 同伴（如果在场）：「……那个影子，比他的身体大。」

叙事含义： 这个矿工正在变成网络的节点——他的"存在"已经不再局限于身体的范围。影子比身体大，因为他在网络中的"体积"比在物理世界中的大。

支线2：孢子平台（Spore Platform）

位置与入口

D 区峡谷和 E 区 Boss 场地之间，一条被苔藓半掩的岔路。岔路入口不显眼——只有当玩家靠近时，蘑菇朝向会微微转向岔路方向（0.5 秒的动画，玩家可能注意到，也可能错过）。

辅助提示： 如果同伴在场，D 区同伴说出「我感觉……有人在摸我的后脑勺」后，同伴的视线会朝岔路方向偏转一次。

空间

5×5m 的圆形平台，被浓密的孢子云完全覆盖。玩家进入后，能见度降至 3 格。平台中央有一块平坦的石头，石头上坐着一个冥想者。

冥想者

  • 外观：菌丝宿主的外观，但姿态是盘腿坐着，双手放在膝盖上。
  • 行为：不攻击。 即使玩家走到它面前，它也不动。不回应交互。
  • 如果玩家攻击它：它不反击。受到第一次伤害后"消散"——像被风吹散的孢子。留下一块铭牌，日期是今天。
  • 如果玩家听完文本后离开，再回来：它仍在原地，说：「你还在。很好。」
  • 如果玩家击败 Boss 后带着晶核回来：冥想者沉默。菌丝微微发光，和背包里晶核的脉动同步。不需要额外文本——纯视觉同步。（判断条件：HasCrystalItem 布尔值）

淡金色文本

玩家进入孢子平台后，不需要按任何键。文本自动出现在屏幕底部，每 10 秒一段。颜色为淡金色，前缀带 ·（冥想者的"自述"标记）。与 E 区 Boss 孢子云记忆文本区分——孢子云文本前缀为淡金色下划线（“网络的记录"标记）。同一 Widget，不同参数。

触发条件 文本
进入平台后 3 秒 · 「你闻到了……铁锈？不，是矿场。你从没来过这个矿场。」
停留 10 秒 · 「这里有人喊过你的名字。但不是你的同伴。」
停留 20 秒 · 「你走出去了。然后你又回来了。这是第三次。」

第三段文本的三种读法：

  • 首次玩家： 困惑——“第三次？我第一次玩啊。”
  • 二周目玩家： 震惊——“它知道。它一直知道。”
  • 仔细阅读文档的人： 这是"庞加莱回归"的最小可行暗示——巨兽的梦中，状态会反复。你已经在巨兽的梦里走过三次。

叙事功能

  • 这是 Demo 中最"不安全"的空间——没有战斗，但信息密度最高，脆弱感最重
  • 将"多周目"从文档设定变成可感知的体验。即使只玩一次，也会感到"有什么地方不对"

真菌之主的叙事身份

它是第一滴源脉之血凝固后、与地底真菌结合而成的原始意识。比螺旋文明更古老。在人类、精灵、矮人出现之前——它就在地底沉睡。没有语言、没有思想——只有一种缓慢的、跨越千年的感知。它记得巨兽的心跳。

矿工触发了什么： 矿工挖到了一片穿透真菌之主沉睡层的矿脉。在这个深度，矿石和菌丝互溶。触碰矿石 = 触碰它。矿工开始异化，不是被寄生——是被吸入它的记忆网络。

真菌之主 vs 矿工的关系： 真菌之主只有一个。矿工不会变成新的真菌之主——他们变成网络的节点。他们的记忆还在，他们通过真菌之主继续存在。BOSS 战不是与一只怪物的战斗——是与一个由矿工记忆编织成的存在战斗。

击败的意义： 击败不是消灭——是暂时切断物理锚点。它在更深处继续存在。晶核是它的一块记忆碎片——它在脉动，在被带出森林后继续"记得"你。

铭牌描述

  • 普通铭牌（统一描述，11 块全部相同）： 一块矿工身份牌。名字已模糊不清。
  • Boss 掉落的铭牌： 一块矿工身份牌。和其他十一块是同一个模具。不同的是——上面的名字被刮掉了。不是锈，是有人用东西刮的。触碰到时，微微发热。
  • 同伴脖子上的铭牌（B 区营救时的 UI 提示）： 他胸前挂着一块金属牌子，生了锈，上面隐约刻着一个名字。

设计要点

  1. 铭牌不区分名字——不需要给每块单独起名。名字模糊意味着"他们是谁不重要，重要的是数字对不上”
  2. C 区菌丝宿主掉落铭牌 → 玩家第一次将"捡铭牌"和"打菌丝宿主"联系起来：“铭牌 = 矿工 → 矿工 = 节点”
  3. Boss 掉落的铭牌是唯一特殊的——名字被刮掉 + 发热。暗示这块铭牌的主人比这十二人更早
  4. 同伴脖子上的铭牌不可拾取——只在 B 区营救时以 UI 提示形式出现一次。F 区同伴说出时，玩家会想起那一瞬间

菌丝网络的场景展现

矿工们没有消失——他们的身体被纳入了菌丝网络。以下是四个展现点，用菌丝、蘑菇和铭牌来暗示身体的存在和转化：

B 区：菌丝覆盖的身形

帐篷外，地面上有一团菌丝覆盖的隆起。菌丝的走势隐约勾出一个趴着的身形——肩膀、脊椎、向一侧弯折的膝盖。旁边的菌丝里半埋着一块铭牌。

C 区：菌丝宿主（已有）

战斗击杀 → 不倒地 → 孢子释放 → 躯体萎缩成干壳 → 2 秒后干壳上长出小蘑菇

D 区：岩壁菌丝脉络

峡谷岩壁上，菌丝沿着裂缝蔓延——不是附着，是嵌入。像石头自己长出了血管。几块铭牌被菌丝穿过——穿过那个刻了名字的位置。铭牌和菌丝已经长在一起了。

E 区：菌丝网络节点

场地中央的地面，所有菌丝脉络在这里收束。交汇处有几个浅浅的凹陷，大小和人形接近。菌丝在这些凹陷里长得最密——像是这里曾经有过什么，菌丝不舍得让它空着。

    - 保底触发：Boss 低血量菌丝网络震荡，菌丝从地面涌入矮人体内——更快更暴力

  矮人头像框边缘出现菌丝纹路。
  他转身，动作帧从"攻击"切成"朝向玩家"。
  最后一句话：
  → "来吧，加入我/我们。"

  矮人暂时成为敌人单位，但比普通敌人弱：
  - HP 只有正常状态的 50%
  - AI 只用基础普攻，不用技能
  - 动作有延迟（菌丝操控 + 他在内心抵抗）

  玩家瞬间变成 1 vs 4（真菌之主 + 2 小怪 + 虚弱矮人）
  压力增加，但不是碾压——矮人故意被设计成虚弱状态

阶段三：释放（~20s）
  矮人的 HP 降到 0 → 不放"死亡红牌" → 触发特殊演出：

  菌丝出体 → 孢子散入空中 → 矮人倒地 → 起身，状态清除

  矮人跪在场地边缘，单手撑地。不再参战。
  他说：
  → "我看到了他们……哈洛德、老詹姆、卡里克……
      他们回不来了。"

  释放效果：
  → 真菌之主短暂失神 1 轮（失去了一个节点）
  → 玩家获得"觉醒"buff（攻击力 +30%，持续 2 轮）

阶段四：收尾（~70s）
  玩家 + buff vs 真菌之主 + 残余小怪
  矮人在边缘，活着，但不再参战。
  击败真菌之主 → 晶核掉落 + 一块铭牌（名字被刮掉，微微发热）

设计原理

为什么是"反水→释放→增益”？

  • 同伴反水时是虚弱的 → 数值上可控（HP 减半 + 无技能 + 动作延迟）
  • 击败他不是杀他 → 是"释放"他 → 和 C 区菌丝宿主的死亡规则一致
  • 释放效果帮助玩家 → 不是惩罚，而是奖励玩家"在乎同伴"
  • Buff 不是无中生有 → 来自真菌之主失去节点时的震荡

为什么不用能量条/大招系统？

  • Demo 开发时间紧，不做 5 分钟才会用一次的新系统
  • Boss 战才第一次用 → 玩家没学会就死了 → 挫败感

为什么不是纯剧情杀？

  • 玩家不能亲手打同伴的话，反水没有情感重量
  • 情感难度 > 数值难度——这是"你必须亲手打败你救的人"

          玩家独自进入森林。

          **出生点前方 8m，一棵倒下的蘑菇。**

            - 精灵学者：「它在……转向？不，是生长方向。生长方向不会错。」
            - 自定义：（没有语音，只有底座阴影闪烁 + 轻微 UI 音效）

B 走廊 ── 营地残骸 + 矿工日记 + 菌丝覆盖的身形。
          「第三天。蘑菇在跟我说话……它似乎在……记住我。」（字迹中断）

          提示文本："他胸前挂着一块金属牌子，生了锈，上面隐约刻着一个名字。"

          同伴站直了，从地上捡起一块石头在手里掂了掂——像在确认自己还活着。沉默 2 秒。
          →「还有……六个人在里面。」

C 空地 ── 两个菌丝宿主（被寄生的矿工）。
          同伴参与战斗，协助清小怪。
          战斗 —— AI 不躲避，朝玩家走来。

          **同伴攻击菌丝宿主时，插入 0.3 秒"识别"动画。**

          攻击音效延迟 0.1 秒——他在犹豫。

          触碰到时，微微发热。」

边缘情况与体验补丁

以下是设计审查中发现的需要补充的边界情况。属于表现层面问题，可在开发中途逐步补齐，不阻塞白盒搭建。

补丁一：同伴死亡分支

当前叙事流程假设同伴全程存活。需设计一条"同伴在 C 区战死"的降级路径。

触发条件

C 区小型遭遇战中，同伴 HP 降至 0 → 不触发"释放"演出（那是 E 区专用）→ 同伴倒地，死亡红牌拍下。玩家可以继续战斗，战斗结算不变。

叙事降级

区域 同伴存活 同伴死亡
D 区 同伴：「我感觉……有人在摸我的后脑勺。」 跳过同伴台词。玩家独自走峡谷。菌丝纹路的光脉节奏比正常快半拍——像少了什么在吸收注意力。
E 区 四阶段完整（并肩→反水→释放→收尾） 跳过反水和释放阶段。直接 1 vs 真菌之主 + 2 小怪。Boss 血量下调 20%（少了同伴输出 + 少了释放 buff）。Boss 击败后晶核不变，但在 Boss 沉入地下后，玩家脚边除了晶核还多了一块铭牌——和 Boss 掉落那块位置不同，更靠近场地边缘，像被丢下的。
F 区 同伴指证委托人。 跳过同伴指证。委托人：「你是第二个回来的。第一个……记不太清了。」数量从"第三个"降为"第二个”——暗示同伴算第一个回来的人，但委托人忘了。微热的矿石仍然给玩家。

设计原则

  • 同伴死亡不惩罚玩家。Boss 血量下调 + 直接跳过反水阶段，整体难度持平甚至略低
  • 叙事不降级为"无趣"——同伴不在场时，D/E/F 区用环境细节填补缺失的台词
  • E 区多出的那块铭牌不计数（铭牌总数仍为 11+1+1=13）。它只是静静地在那里

补丁二：战斗教学触发器

A 区只教了自由探索移动（WASD）。C 区是第一场真正的遭遇战，中间缺少回合制战斗的教学桥。

方案：A 区出口前的"不会反击的敌人"

在 A 区即将进入 B 区的通道处（A 区边缘，玩家必经之路上），放置一只 孢子虫：

属性 内容
名称 孢子虫（Spore Mite）
外观 复用小型蘑菇模型，缩小至 0.3 倍，加一个缓慢上下浮动的动画。不需要新 Sprite
AI 无。不移动，不攻击。固定位置
HP 1
触发方式 玩家靠近到 3m 时弹出半透明 UI 提示

教学文本（分两段弹出）

第一段（靠近时）：

前方有一只孢子虫。按 Tab 进入战斗模式。

第二段（进入战斗后，玩家完成第一次移动）：

选择一个格子移动。靠近目标后，点击它进行攻击。

击败后：

  • 孢子虫萎缩（复用菌丝宿主死亡演出的简化版——只萎缩不长蘑菇）
  • 掉落一块铭牌（让玩家建立"战斗掉落 → 铭牌"的认知）
  • 铭牌描述和 B 区普通铭牌一致：一块矿工身份牌。名字已模糊不清。

设计要点

  • 不打断沉浸感。孢子虫是"世界里的生物"，不是"教学 NPC"
  • 不掉落不消失——如果玩家忽略它直接走进 B 区，它只是静静地浮动在那里
  • 铭牌总数从 11 拾取变为 12 拾取。但 F 区结尾文本不改——“你数了数铭牌。十二块。“教学战掉落的那块就是第十二块
  • 10 秒完成，不拖节奏

补丁五：玩家返回已通过区域的行为

线性珠串结构不锁死已通过区域的通道。玩家可以在任意时刻回头走。需要定义回退行为，避免出 bug。

规则

行为 规则
已拾取的铭牌 不重新生成。拾取位置保留空地面 Decal
已击败的菌丝宿主 干壳保留在原地（不消失）。蘑菇继续长在干壳上
已解救的同伴 跟随玩家。如果玩家从 C 区走回 B 区，同伴跟着走，不在营地处重新出现
已触发的叙事事件 日记不再弹出（已读过）。异常蘑菇不再触发底座阴影闪烁（已触发过）
已进入的 Boss 场地 Boss 战触发后 E 区入口不锁。但如果玩家在 Boss 战胜利后走回 E 区，只看到空的菌丝节点场地——菌丝不再发光，孢子云消散。地上的干壳还在
委托人 A 区和 F 区是同一个 NPC。如果玩家从 B/C/D 回到 A 区，委托人站在原地，只有一句通用台词：「调查进展如何？」（无剧情推进）

不需要的内容

  • 不需要回头路对话。不需要"你确定要往回走吗"提示。不需要隐藏区域
  • 目标是：回头走不出 bug，但不鼓励回头走

补丁六：结束画面的技术实现

当前描述：“黑屏 3 秒 → 手写体文本 → Steam 愿望单画面”。以下是技术落地细节。

实现方案

黑屏： Widget 全屏黑色背景，使用 UWidgetAnimation 做淡入（0→100% opacity，1.5s），而不使用关卡加载/卸载。保留当前关卡在背后渲染（性能无影响——F 区是 10m 小径，场景极简）。

手写体文本： 便签材质（复用现有 M_UI_StickyNote 材质实例），分三行显示，逐行淡入：

备选方案（更低成本）

如果 UI 动画开发时间不够：直接切成一张全屏静态图片（PNG），包含手写体文本 + 暗色渐变背景。玩家按任意键后显示愿望单页面。牺牲"逐行淡入"的效果，但不牺牲文本本身。

60 分钟版本的核心卖点

“一场 60 分钟的完整遭遇——每次偏离主线的探索，都会让你离真相更近一步，但也离安全更远一步。”

这不是"一个更长的 Demo"。这是一个有深度选择的短篇体验。玩家在 60 分钟内会经历：

  1. 发现不对劲（A 区蘑菇）
  2. 面对"真相不是别人给的"（回声裂隙——矿工自己在画螺旋纹路）
  3. 建立情感连接（B 区营救同伴）
  4. 面对"你已经被认出了"（孢子平台——冥想者知道你是第几次来）
  5. 亲手打朋友（E 区反水）
  6. 带走一个问题 + 一个谎言（F 区——委托人说"第三个"，但铭牌数不对；铭牌计数器全程在说谎）

30 分钟版本展示异常——这个世界不对。60 分钟版本让你亲手触碰真相——而每一次触碰，都在改变你。你不是在"发现"这个世界的秘密，你是在"选择"知道多少。而你知道得越多，身体越开始变得不像自己。

60 分钟版本是 Project_DND 的"完整体验 Demo"。它不是更长——而是更深。每一次偏离主线的选择，都是玩家自愿支付的"认知的代价"。
