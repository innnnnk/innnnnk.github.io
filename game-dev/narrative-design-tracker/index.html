贯穿性设计原则（影响所有模块）

这些原则不是某个模块的子项——它们是渗透在全部五个模块中的设计基因。每一个模块的每一项决策都应遵循这些原则。

原则 内容 状态
冰山原则 前期绝不出现高维/体制/空性等词汇。一切用最接地气的凡人情感包装。表面是派系斗争、矿石争夺、复仇与背叛——深层才是宇宙法则 ✅ 已确定
单源怪诞 所有异常现象统一来源：源脉矿石接触 + 时间 = 异化。不"为了怪而怪"——每个怪诞都有统一的解释引擎 ✅ 已确定
三层认知阶梯 第一幕：正常DND奇幻 → 第二幕：世界不对劲 → 第三幕：原来……。Demo 在第一幕，只埋种子 ✅ 已确定
三个核心母题 「记得」— 贯穿声音/文本/NPC；「这里是活的」— 贯穿声音/视觉/玩法；「这里不完全是我们的」— 地质深处埋着不属于任何已知文明的东西 ✅ 已确定
微克苏鲁 手法：用克苏鲁的"对未知的恐惧"作为叙述手段，指向的是"你有你不理解的东西埋在自己身上"——不是外部的不可名状，是内部的遗忘。内核：最终落点是万炁同源——恐惧是引子，认知是过程，回家是结局。详见下方展开 ✅ 已确定

叙事引导：七条通道

这是 Project_DND 的通用叙事方法论——不仅适用于真菌森林，也适用于正式版所有关卡。七条通道共享两个核心母题：

  • 「记得」 — 贯穿声音、文本、NPC、玩法
  • 「这里是活的」 — 贯穿声音、视觉、玩法

通道一：声音引导

声音不说任何话，但让身体先于大脑感到不安。

原则： 嗡鸣的频率和人紧张时体内的共振接近。声音只在关键时刻变化——变化本身就是叙事。

位置原型 声音设计 引导什么
全局 环境底噪——与环境主题匹配的持续低频声（不是音乐，是场景本身的"呼吸声"） 全程暗示：这个场景不是普通的场景
异常物附近 靠近异常物 → 底噪频率改变（变高/加泛音/短暂消失） 「这里有什么不对劲」
战斗击杀 敌人被击败 → 非人叹息（不是惨叫） 「它不是在死亡，它是在被释放」
BOSS 战后期 底噪变多重和声（多个声音同时低语） 「它不止是一个，它是很多个」
出口 离开场景 → 底噪消失，短暂完全安静，然后自然声音恢复 「你出来了。但那个声音还在你脑子里」

通道二：视觉异常引导

视觉异常都在玩家注意力边缘——不是过场动画，是眼角余光捕捉到的。

原则： 异常要在"看得见但又没完全看见"的区域。它们不说任何话，但组合起来建立一个认知：这里的存在法则，和认知下的不一样。

位置原型 视觉设计 引导什么
营地/遗迹 残留物品上有细微的异常生长（菌丝/结晶/锈蚀不过是在向上蔓延） 「他们离开得很突然。或者根本没离开」
战斗中 敌人死亡 → 不倒地 → 体内"释放"（孢子/光/烟）→ 躯体萎缩成空壳 → 空壳上长出新东西 「死亡规则在这里不一样」
异常符号附近 玩家自己的影子短暂消失/变形/延迟 → 恢复 「你的存在在这里不稳定」
异常符号本身 微弱的脉搏般亮度变化（低频闪烁，不规律） 「它是活的」
BOSS 场地 战斗网格本身偶尔闪烁/错位 「物理法则在这里不稳定」
出口 离开后，场景中的某个元素（植被/石头/光影）微微转向玩家离开的方向 「它们在看着你」

通道三：文本引导（日记 / 物品描述）

文本不解释，只呈现现象。让玩家自己连起来。

原则： 每段文本只说一件事——一个细节、一个感受、一个矛盾的观察。不串联，不总结。字迹中断和反复修改本身就是叙事。

位置原型 文本类型 引导要点
早期区域 日记/记录 引入核心概念（记忆、异常感知等），用凡人视角而非解释性语言
BOSS 战后 掉落物描述 Callback 之前的概念，把 NPC 日记和 BOSS 连起来
可选区域 补充记录 提供额外信息——不解释，只是多一个"不对劲"的例证

通道四：NPC 对话引导

说最少的话，露出最多的疑点。

原则： Demo 所有 NPC 的台词总和应少于一张 A4 纸。每句话都要承担至少两个功能（推进任务 + 暗示异常 / 传递信息 + 制造问题）。

位置原型 谁 功能
入口 委托人/引导NPC 建立任务 + 铺垫矛盾（说的话在结尾会有回响）
探索中 同伴（如有） 提供"内部人"视角——他说的话玩家可能没资格说
BOSS 前/后 同伴/引导NPC 状态变化：感染/清醒/沉默——台词减少本身就是叙事
出口 委托人/同伴 回收入口的铺垫 + 制造新的疑问

通道五：空间节奏引导

空间本身就是叙事。玩家不需要任何文本，光是走一遍，身体就会感受到情绪的起伏。

原则： 通过改变路径的宽度、曲率和围合感来控制情绪。六个区域的典型情绪弧线：

区域位置 空间特征 引导的情绪
入口 开阔、暖光、低密度信息 安全感：「这是一个正常的地方」
过渡 S 形弯道、逐渐变冷的光 好奇 + 轻微不安：「拐角后面是什么」
遭遇前 突然开阔、但被障碍物分割 警觉：「这里可能有东西」→ 战斗
遭遇后 收窄、暗光 压抑：「我必须往前走，但我不确定我想」
BOSS 圆形封闭、强对比光 被困感：「无路可退」
出口 重新开阔、暖光 释放：「我出来了」——但不安还在

通道六：光渐变引导

光的变化速度要慢到玩家不会主动注意到——他们只会感觉"怎么越来越不舒服"但说不出为什么。

原则： 完整的光色弧线：暖（安全）→ 冷（异界）→ 最暗（威胁峰值）→ 戏剧对比（高潮）→ 暖（回归但不完全）。

阶段 光色 引导什么
入口 → 过渡 暖 → 冷渐显 从「安全的人类世界」过渡到「异常场域」
过渡 → 遭遇 均匀中性 让玩家专注于战斗，暂时忘记环境异常
遭遇 → BOSS 前 光照降到全局最暗 视觉上的威胁感达到峰值
BOSS 战 戏剧化强对比（亮处极亮，暗处极暗） 史诗感——这是最终对决
BOSS → 出口 暖金色胜利光 让玩家在物理和心理上同时回归「安全」——但不完全

通道七：玩法反馈引导（「玩法 = 叙事」连接点）

玩法是唯一一种玩家「必须参与」的叙事。看日记可以跳过，听声音可以忽略——但战斗中发生的事情，每个玩家都会经历。

原则： 永远不问玩家「你怎么看」，只给他们一个必须参与的情境，而他们在其中不得不产生一个反应。

位置原型 玩法机制 叙事引导
探索中 异常物可交互，靠近时战斗网格短暂闪烁 「矿石和空间稳定性有关」——具体为什么，不解释
遭遇战 敌人的 AI 行为异常：不躲避、不逃跑——朝玩家走来，像在拥抱 「它们不是在攻击你。它们想传染你。」
BOSS 战 掩体（孢子云/暗影等）提供战术优势——但站太久会被缓慢感染 「掩体不是安全的。它在改变你。」
全局 击杀一只后，下一只在战斗开始时多看你半秒 「它认识你。或者它认识你身上的某个东西。」

声音说： 「不舒服。」         —— 嗡鸣、叹息、消失

空间说： 「你越来越深入了。」 —— 收窄、变暗、围合
光说：   「你离开安全区域了。」 —— 暖→冷→暖
玩法说： 「规则不一样。」     —— 死亡≠死亡、感染是诱人的

玩家不会分解出「这是声音引导、这是文本引导」。他们只会有一种感觉：

「这个地方不对劲。我要继续探索。」

已知的参照物 + 一个无法解释的偏离 = 未知的恐惧

传统克苏鲁：
"你打开门。门后是一双不属于任何生物的眼睛。你疯了。"

本作的微克苏鲁 + 万炁同源：
"你打开门。门后是一双你认识的眼睛——你自己的。
但它不在你脸上。它在某个更大的东西上。
你现在知道了一件事——你不是你。
你没有被吓疯。你被悲伤淹没了。"

核心规则

所有微克苏鲁暗示必须遵循一个关键规则：不能指向"外部威胁"，只能指向"你和你的关系"。

❌ 指向外部（破坏万炁同源） ✅ 指向自身（兼容万炁同源的前半程体验）
“有个东西在森林里看着你” “你走出森林后，发现你看森林的方式变了”
“存在一个超出理解范围的怪物” “超出理解范围的是你和世界的关系——蘑菇记得你，你不知道为什么”
“这个世界是假的，外面有真的” “这个世界是真的。你以为是独立个体的自己——可能不是”

第一幕/Demo（微克苏鲁表层）：
  日记字迹中断 → 书是空白的 → 你读到的记忆是谁的？
  菌丝宿主认识你 → 你不认识它 → 你怎么可能认识它？

第二幕/正式版 Act 2（过渡）：
  不是怪物在追你 → 是你自己在追自己
  源脉矿石是巨兽的血 → 你不是在探索世界 → 你是在探索你自己
  玩家的感受："我在理解这里的规则……但理解本身让我不舒服。"（认知 = 过程）

第三幕/正式版 Act 3（万炁同源揭示）：
  那个不可名状的存在 = 是你自己，是更大的你
  不是宇宙冷漠 → 是宇宙在等你回家
  self.immune_response() — 你最后的敌人不是怪物，是你的分离恐惧
  玩家的感受："原来……"（回家 = 结局）
