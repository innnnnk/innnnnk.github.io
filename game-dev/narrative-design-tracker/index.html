贯穿性设计原则（影响所有模块）

这些原则不是某个模块的子项——它们是渗透在全部五个模块中的设计基因。每一个模块的每一项决策都应遵循这些原则。

原则 内容 状态
冰山原则 前期绝不出现高维/体制/空性等词汇。一切用最接地气的凡人情感包装。表面是派系斗争、矿石争夺、复仇与背叛——深层才是宇宙法则 ✅ 已确定
单源怪诞 所有异常现象统一来源：源脉矿石接触 + 时间 = 异化。不"为了怪而怪"——每个怪诞都有统一的解释引擎 ✅ 已确定
三层认知阶梯 第一幕：正常DND奇幻 → 第二幕：世界不对劲 → 第三幕：原来……。Demo 在第一幕，只埋种子 ✅ 已确定
三个核心母题 「记得」— 贯穿声音/文本/NPC；「这里是活的」— 贯穿声音/视觉/玩法；「这里不完全是我们的」— 地质深处埋着不属于任何已知文明的东西 ✅ 已确定
微克苏鲁 手法：用克苏鲁的"对未知的恐惧"作为叙述手段，指向的是"你有你不理解的东西埋在自己身上"——不是外部的不可名状，是内部的遗忘。内核：最终落点是万炁同源——恐惧是引子，认知是过程，回家是结局。详见下方展开 ✅ 已确定

真菌之主的叙事身份

它是第一滴源脉之血凝固后、与地底真菌结合而成的原始意识。比螺旋文明更古老。在人类、精灵、矮人出现之前——它就在地底沉睡。没有语言、没有思想——只有一种缓慢的、跨越千年的感知。它记得巨兽的心跳。

矿工触发了什么： 矿工挖到了一片穿透真菌之主沉睡层的矿脉。在这个深度，矿石和菌丝互溶。触碰矿石 = 触碰它。矿工开始异化，不是被寄生——是被吸入它的记忆网络。

真菌之主 vs 矿工的关系： 真菌之主只有一个。矿工不会变成新的真菌之主——他们变成网络的节点。他们的记忆还在，他们通过真菌之主继续存在。真菌之主通过他们"看"到了人类。Demo 里玩家击败两个菌丝宿主——他们在那一刻被暂时释放，但记忆留在了网里。BOSS 战不是与一只怪物的战斗——是与一个由矿工记忆编织成的存在战斗。

击败的意义： 击败不是消灭——是暂时切断物理锚点。它在更深处继续存在。晶核是它的一块记忆碎片——它在脉动，在被带出森林后继续"记得"你。

微克苏鲁设计原则

核心公式 

1

已知的参照物 + 一个无法解释的偏离 = 未知的恐惧

不是展示怪物——是展示一个你认识的东西，但它有一个细节不对，而那个细节意味着你所有认知都可能不成立。

与万炁同源的关系

传统克苏鲁 本作的微克苏鲁
恐惧指向 外部的不可名状存在 内部的遗忘 — 你身上有你不理解的东西
人类的位置 渺小、无意义 不是渺小 — 你是更大的存在的碎片，只是忘了
宇宙的态度 冷漠、不可理解 不是冷漠 — 是你目前无法理解。认知的代价是理解后不再是"你"
结局质地 绝望、疯狂 不是绝望 — 是回家。第三幕的最终选择不是胜利或失败，是融合或分离

传统克苏鲁：
"你打开门。门后是一双不属于任何生物的眼睛。你疯了。"

本作的微克苏鲁 + 万炁同源：
"你打开门。门后是一双你认识的眼睛——你自己的。
但它不在你脸上。它在某个更大的东西上。
你现在知道了一件事——你不是你。
你没有被吓疯。你被悲伤淹没了。"

Demo 中的具体表现

所有微克苏鲁暗示必须遵循一个关键规则：不能指向"外部威胁"，只能指向"你和你的关系"。

❌ 指向外部（破坏万炁同源） ✅ 指向自身（兼容万炁同源的前半程体验）
“有个东西在森林里看着你” “你走出森林后，发现你看森林的方式变了”
“存在一个超出理解范围的怪物” “超出理解范围的是你和世界的关系——蘑菇记得你，你不知道为什么”
“这个世界是假的，外面有真的” “这个世界是真的。你以为是独立个体的自己——可能不是”
