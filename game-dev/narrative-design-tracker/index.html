CRPG 剧情设计模块全景表

这是一份活文档——记录 Project_DND 剧情设计五个模块的当前状态。每当我们确定一个模块的子项，就更新这张表。Demo 阶段只需要从每个模块中取最小单元。

📐 五个模块的依赖关系：地基 → 结构 → 人物 → 具体内容 → 体验设计
地基没定，上层全部悬空。人物没定，文本没法写。体验设计贯穿全部层级。

第一层：地基（世界观设定之前就要想清的）

子模块核心问题当前状态结论
核心冲突这个世界最根本的矛盾是什么？谁和谁因为什么不可调和？✅ 已确定三层嵌套：表层（Demo用）— 认知分歧：矿工说矿石在说话，公会说这是精神病；中层（正式版 Act 2）— 生存伦理：文明需要矿石能源 vs 矿石接触会让人异化；深层（正式版 Act 3）— 自我瓦解：异化的本质是"想起了自己不是独立个体"
主题这个故事到底在讲什么？不是情节，是"关于什么"✅ 已确定“认知的代价” — 每深一层都要付出情感/身体代价。在这个世界里，理解 = 异化。知道得越多，你的身体越不像你自己
情感基调悲壮？荒诞？忧伤？希望？✅ 已确定微妙的不对劲 — 越深入越稀薄/抽象（Backroom式衰减）。不是恐怖，不是血腥，是"少了什么"。浅层：有缺失的正常。深层：有信号的空白
玩家角色锚玩家是"谁"？✅ 已确定佣兵公会调查员 — Demo 三选一快速职业选择（战士/法师/游荡者）。表层身份具体，底层埋着"被认出"的种子（回声藓用你的声音说话、影子短暂消失）
怪诞法则怪诞的规则是什么？✅ 已确定单源怪诞 — 全部源于源脉矿石接触 + 时间 = 异化。分三级：L1 认知偏移（Demo大量用）、L2 生物形态（Demo有限用）、L3 逻辑崩塌（正式版第三幕用）。怪诞随剧情深入而加剧——越接近巨兽核心，梦越不稳定

第二层：结构（故事怎么讲）

子模块核心问题当前状态结论
三幕/章节大纲故事分几段？每段玩家目标是什么？⚠️ 世界观文档有三幕认知阶梯，但情节大纲未设计世界观揭示阶梯 ≠ 情节推进阶梯。两者需要叠加。正式版章节大纲待推进
关卡叙事映射（Demo）真菌森林 6 区各自承载什么叙事功能？✅ 已确定A 入口 — 安全感建立 + 任务接收；B 走廊 — 好奇 + 第一个异常（日记）；C 空地 — 警觉→战斗 + 死亡规则展示；D 峡谷 — 压抑 + 古老存在暗示；E 竞技场 — 被困感 + BOSS 战 + 怪诞升级；F 出口 — 释放 + 核心疑问（委托人记忆矛盾）
关键路径（Demo）Demo 玩家必须经历的事件序列✅ 已确定线性珠串：A→B→C→D→E→F。无分支
分支点玩家的选择在哪里改变故事走向？❌ Demo 不需要正式版待设计
节奏控制（Demo）紧张/舒缓交替✅ 已确定通过空间节奏（宽度/曲率/围合感）+ 光渐变（暖→冷→暖）+ 怪诞梯度（0→7/10）三重控制。详见 真菌森林展示场景

第五层：体验设计（玩家感受到什么）

子模块核心问题当前状态结论
入门钩子（Demo）前 3-5 分钟抓住玩家✅ 已确定不是意象化开场 — 是具体的目标。玩家看到森林入口、接到委托、选职业、走进去。第一个 5 分钟：熟悉操作 + 感受空间 + 发现第一本日记
Demo 结束钩子Demo 结束后玩家脑子里留着什么✅ 已确定“你是第三个回来的。前两个……记不太清了。” + 包裹里那块微热的矿石 + BOSS 的脉动晶核。玩家带走的不是答案，是一个问题：“这是什么地方？”
Demo 结束画面Steam 新品节 Demo 结束后的呈现⚠️ 待定考虑结束画面文字：「你走出森林。包裹里的晶石还是热的。」+ 愿望单链接
多周目/NG+ 设计庞加莱回归、文本微调⚠️ 世界观文档有方向Demo 不涉及。正式版数据驱动实现方案已讨论
Demo 硬约束（不做什么）防止 scope creep✅ 已确定不解释源脉矿石起源、不展示真菌教派真人、不揭示巨兽之梦、不让 NPC 说出任何世界观关键词、不做超过 10 秒的过场动画、不做存读档
